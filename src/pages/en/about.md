## Introduction

A game server developer with over 20 years of experience in game server platform design and development.

I have designed and implemented server platforms for various genres, from real-time games to web-based games, with particular expertise in building high-performance TCP/IP-based network systems.

Proficient in multiple programming languages including C++, C#, Java, and Kotlin, with deep understanding of Spring and .NET Framework to select and implement optimal technology stacks.
